# Books-app

Single page application for searching books from Google Books APIs.

## Deploy:

### [Books Application](https://books-app-alexk08.netlify.app/)

## Stack

- React v18
- MobX
- Typescript
- Axios
- TailwindCSS & DaisyUI
- Create React App
- React Router v6.4.3
- [Google Books APIs](https://developers.google.com/books)

## Usage

### `npm i`

### `npm start`

### `npm build`
